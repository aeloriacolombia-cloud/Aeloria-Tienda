# Aeloria Tienda

Sitio web desarrollado para la tienda Aeloria, especializada en productos de cuidado de la piel y belleza.

## Descripción

Aeloria Tienda es una página web diseñada para mostrar diferentes categorías de productos de skincare y belleza, ofreciendo una experiencia sencilla y agradable para los usuarios.

## Características

- Página principal de bienvenida.
- Catálogo de productos.
- Categorías organizadas.
- Sección de skincare.
- Sección de cuidado facial.
- Diseño responsivo.
- Estilos personalizados con CSS.

## Tecnologías utilizadas

- HTML5
- CSS3
- GitHub
- GitHub Pages
